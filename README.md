~~~~~~~~~~~~~~~~~~~~
pip install lazypsf
~~~~~~~~~~~~~~~~~~~~

To get to grips with how to use lazypsf [See tutorial](https://github.com/ryanc123/LazyPSF/blob/master/Tutorial/lzypsf_tut.ipynb)
